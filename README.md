# Breast Cancer Diagnosis Predictor

## Project Overview

This Machine Learning project predicts whether a breast tumor is **Benign** or **Malignant** using the Breast Cancer Wisconsin Diagnostic Dataset.

The model is trained using a Random Forest Classifier and deployed with Streamlit.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Matplotlib
* Seaborn
* Joblib

## Model Performance

* Accuracy: 96.49%

## Features

* Breast cancer prediction
* Interactive web application
* Feature importance analysis
* Data visualization
* Prediction confidence score

## Project Files

* app.py
* train.py
* eda.py
* feature_importance.py
* model.pkl

## How to Run

Install dependencies:

pip install pandas numpy scikit-learn matplotlib seaborn streamlit joblib

Train the model:

python train.py

Run the application:

python -m streamlit run app.py
